Sobre o Projeto

Sistema completo de D&D 5e com foco em conteúdo homebrew. Desenvolvido para facilitar a criação e gerenciamento de personagens, com interface intuitiva e recursos avançados para jogadores e mestres.

Todas as fichas são salvas localmente no seu navegador e podem ser exportadas como arquivos JSON para backup ou compartilhamento.
